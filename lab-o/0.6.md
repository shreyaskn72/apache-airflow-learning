Installing **Apache Airflow** on Windows 11 directly is *not recommended* because Airflow is designed for Unix-like systems. But you can still run it smoothly using **WSL (Windows Subsystem for Linux)** or Docker.

I’ll give you the **best approach (WSL)** and a simpler alternative (Docker).

---

# ✅ Option 1: Install Airflow using WSL (Recommended)

## Step 1: Install WSL

Open PowerShell (Admin):

```bash
wsl --install
```

This installs **Ubuntu** by default.

Restart your system.

---

## Step 2: Open Ubuntu (WSL)

Search for **Ubuntu** in Start Menu and open it.

Update packages:

```bash
sudo apt update && sudo apt upgrade -y
```

---

## Step 3: Install Python & pip

Airflow works best with Python 3.8–3.11.

```bash
sudo apt install python3-pip python3-venv -y
```

---

## Step 4: Create Virtual Environment

```bash
mkdir airflow-project
cd airflow-project

python3 -m venv airflow-venv
source airflow-venv/bin/activate
```

---

## Step 5: Install Airflow

Set Airflow home:

```bash
export AIRFLOW_HOME=~/airflow
```

Install Airflow (example version):

```bash
pip install "apache-airflow==2.8.1" --constraint "https://raw.githubusercontent.com/apache/airflow/constraints-2.8.1/constraints-3.10.txt"
```

---

## Step 6: Initialize Database

```bash
airflow db init
```

---

## Step 7: Create Admin User

```bash
airflow users create \
  --username admin \
  --firstname Admin \
  --lastname User \
  --role Admin \
  --email admin@example.com \
  --password admin
```

---

## Step 8: Start Airflow

Run these in **two terminals**:

### Terminal 1:

```bash
airflow webserver --port 8080
```

### Terminal 2:

```bash
airflow scheduler
```

---

## Step 9: Access UI

Open browser:

```
http://localhost:8080
```

Login with your credentials 🎉

---

# 🐳 Option 2: Install using Docker (Easier setup)

If you're okay with containers, use **Docker**.

## Steps:

1. Install Docker Desktop (enable WSL integration)
2. Run:

```bash
curl -LfO 'https://airflow.apache.org/docs/apache-airflow/stable/docker-compose.yaml'
docker-compose up
```

3. Open:

```
http://localhost:8080
```

---

# ⚠️ Important Notes

* ❌ Don’t install Airflow directly via Windows CMD/Python
* ✅ Always use WSL or Docker
* 💡 Use SQLite for learning, PostgreSQL/MySQL for production

---
